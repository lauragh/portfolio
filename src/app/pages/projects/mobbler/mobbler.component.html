Mobbler

Proyecto ABP (aprendizaje basado en proyectos) realizado en 4º de carrera. Aplicación basada en diseño de interiores 3D para empresas dedicadas a la venta de mobiliario.
Rol: FullStack
Tareas:

  • Diseño de mockups (Figma)
  • Implementación de interfaces
  • Desarrollo de un chatbot (RASA)
  • Modelado 3D (Blender)
  • Motor gráfico (Threejs)
  • Creación de script generador datos sintéticos
  • Conexión de la base de datos y API REST
  • Desarrollo de dashboards (PowerBI)
